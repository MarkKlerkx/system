# Setup Docker Server
Script to setup Docker on an Ubuntu 26.04 server. Run the commands below on your server:  

cd ~  
mkdir scripts  
cd ~/scripts  
wget https://raw.githubusercontent.com/MarkKlerkx/system/refs/heads/main/setup_docker_server.sh  
sudo chmod +x setup_docker_server.sh  
sudo ./setup_docker_server.sh
